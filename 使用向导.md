# 微信公众号文章转 Markdown - 使用向导

> 📖 3 分钟快速上手，轻松将微信公众号文章转换为 Markdown 格式

## 🎯 这个工具能做什么？

简单来说，这个工具可以帮你：
- ✅ 把微信公众号文章变成 Markdown 格式
- ✅ 自动下载文章里的所有图片
- ✅ 打包成文件方便保存和分享
- ✅ 将 Markdown 内容转换为精美的流光卡片图片

**不需要注册，不需要登录，完全免费！**

---

## 🚀 快速开始（3 步搞定）

### 第 1 步：打开工具

访问网站后，点击页面上的 **"开始转换"** 按钮，进入转换页面。

### 第 2 步：粘贴文章链接

1. 找到你想转换的微信公众号文章
2. 复制文章链接（链接格式类似：`https://mp.weixin.qq.com/s/xxxxx`）
3. 粘贴到输入框中

**💡 小技巧：** 可以一次添加多个文章链接，点击"添加更多文章"按钮即可。

### 第 3 步：获取并转换

1. 点击 **"获取文章内容"** 按钮
2. 等待几秒钟，系统会自动获取文章内容
3. 在"内容预览"页面确认内容无误后，点击 **"转换为 Markdown"** 按钮
4. 转换完成后，可以选择：
   - 在"下载"页面下载 Markdown 文件或压缩包
   - 在"MD预览"页面导出为精美的流光卡片图片

**完成！** 🎉

## 📝 详细操作指南

### 场景一：转换单篇文章

**适用场景：** 想把一篇好文章保存下来，方便以后查看

**操作步骤：**

1. **输入链接**
   - 在"转换"标签页的输入框中，粘贴微信公众号文章链接
   - 链接格式：`https://mp.weixin.qq.com/s/xxxxx`

2. **获取内容**
   - 点击"获取文章内容"按钮
   - 系统会自动提取文章标题和内容
   - 成功后会自动跳转到"内容预览"页面

3. **预览确认**
   - 在"内容预览"页面查看文章内容
   - 确认内容正确后，点击"转换为 Markdown"按钮

4. **下载文件**
   - 转换完成后，切换到"下载"标签页
   - 选择下载方式：
     - **仅 Markdown**：只下载文字内容（.md 文件）
     - **完整包（含图片）**：下载包含图片的压缩包（.zip 文件）

**推荐：** 如果文章有图片，建议选择"完整包（含图片）"，这样图片会一起打包下载。

5. **导出为图片（可选）**
   - 在"MD预览"页面，点击 **"导出为图片"** 按钮
   - 配置流光卡片参数（模板、样式、尺寸等）
   - 系统会根据内容长度自动判断：
     - 内容较短 → 生成单张图片并下载
     - 内容较长 → 自动分段生成多张图片并打包为 ZIP

---

### 场景二：批量转换多篇文章

**适用场景：** 想把多篇相关文章一次性转换并保存

**操作步骤：**

1. **添加多个链接**
   - 输入第一个文章链接
   - 点击"添加更多文章"按钮
   - 继续输入其他文章链接
   - 可以添加任意多个（建议不超过 10 篇，避免处理时间过长）

2. **批量获取**
   - 点击"获取文章内容"按钮
   - 系统会依次获取所有文章的内容
   - 等待所有文章获取完成

3. **逐个转换**
   - 在"内容预览"页面，每篇文章都有独立的"转换为 Markdown"按钮
   - 点击对应文章的转换按钮
   - 可以一次性转换所有文章，也可以选择性转换

4. **批量下载**
   - 转换完成后，切换到"下载"标签页
   - 每篇文章都可以单独下载
   - 或者点击"下载完整压缩包"，一次性下载所有文章

**💡 小技巧：** 批量下载时，每篇文章会放在独立的文件夹中，方便整理。

---

### 场景三：导出为精美图片

**适用场景：** 想将文章内容转换为精美的图片卡片，用于分享或展示

**操作步骤：**

1. **转换文章**
   - 先完成文章获取和 Markdown 转换
   - 切换到"MD预览"标签页

2. **配置卡片参数**
   - 点击文章卡片上的 **"导出为图片"** 按钮
   - 在弹窗中配置：
     - **模板类型**：选择喜欢的卡片模板（默认模板、金句模板、书摘模板等）
     - **标题和内容**：可以编辑标题和内容
     - **样式设置**：调整宽度、高度、内边距
     - **分辨率**：选择图片清晰度（1x/2x/3x）

3. **导出图片**
   - 点击 **"导出图片"** 按钮
   - 系统会自动判断：
     - 如果内容适合单张图片 → 直接下载 PNG 文件
     - 如果内容过长 → 自动分段生成多张图片并打包为 ZIP

**💡 小技巧：** 
- 调整高度可以控制每张图片的内容量
- 内容较长时会自动分段，无需手动处理
- 多张图片会自动打包，方便管理和分享

---

### 场景四：查看和管理历史记录

**适用场景：** 想重新查看之前转换过的文章

**操作步骤：**

1. **查看历史**
   - 点击左侧导航栏的"历史记录"标签
   - 可以看到所有之前获取或转换过的文章

2. **重新加载**
   - 找到想查看的文章
   - 点击"查看"按钮
   - 文章会自动加载到当前工作区

3. **删除记录**
   - 点击文章卡片上的删除按钮（垃圾桶图标）
   - 可以删除单条记录
   - 或者点击"清除全部"按钮，删除所有历史记录

**💡 提示：** 历史记录保存在浏览器本地，清除浏览器数据会丢失历史记录。

### 下载选项说明

#### 选项 1：仅 Markdown
- **文件格式：** `.md` 文件
- **包含内容：** 纯文本 Markdown 格式
- **适用场景：** 只需要文字内容，不需要图片

#### 选项 2：完整包（含图片）
- **文件格式：** `.zip` 压缩包
- **包含内容：** Markdown 文件 + 所有图片
- **文件结构：**
  ```
  文章标题.md
  images/
    ├── image_1.jpg
    ├── image_2.png
    └── ...
  ```
- **适用场景：** 需要完整保存文章，包括图片

#### 选项 3：批量下载
- **文件格式：** `.zip` 压缩包
- **包含内容：** 所有已转换的文章
- **文件结构：**
  ```
  文章1/
    ├── 文章1.md
    └── images/
  文章2/
    ├── 文章2.md
    └── images/
  ...
  ```
- **适用场景：** 批量转换多篇文章时使用

#### 选项 4：导出为流光卡片图片
- **文件格式：** `.png` 图片文件或 `.zip` 压缩包
- **包含内容：** 
  - 内容较短 → 单张 PNG 图片
  - 内容较长 → 多张 PNG 图片打包为 ZIP
- **特点：**
  - 精美的卡片样式设计
  - 多种模板可选（默认、金句、书摘等）
  - 可自定义尺寸、样式、分辨率
  - 自动智能分段，无需手动处理
- **适用场景：** 
  - 分享文章到社交媒体
  - 制作精美的内容展示图
  - 保存为图片格式便于查看

## ⚠️ 常见问题快速解答

### Q: 为什么获取文章失败？

**可能原因：**
- 链接格式不对（必须是 `https://mp.weixin.qq.com/s/...` 格式）
- 文章链接已失效
- 网络连接问题

**解决方法：**
- 检查链接是否正确
- 尝试在浏览器中打开链接，确认可以访问
- 检查网络连接

---

### Q: 图片下载失败怎么办？

**解决方法：**
- 图片下载失败不影响 Markdown 转换
- 可以选择"仅 Markdown"下载，然后手动添加图片
- 或者稍后重试转换

---

### Q: 转换后的格式不对？

**解决方法：**
- 可以在"MD预览"页面查看效果
- 如果格式有问题，可以手动调整
- 或者使用其他 Markdown 编辑器进一步编辑

---

### Q: 如何导出为图片？

**操作步骤：**
1. 完成文章转换后，切换到"MD预览"标签页
2. 点击文章卡片上的"导出为图片"按钮
3. 在弹窗中配置卡片参数（模板、样式、尺寸等）
4. 点击"导出图片"按钮即可

**说明：**
- 系统会根据内容长度自动判断生成单张或多张图片
- 内容较长时会自动分段并打包为 ZIP
- 可以调整高度来控制每张图片的内容量

---

### Q: 历史记录会保存多久？

**说明：**
- 历史记录保存在浏览器本地
- 最多保存 50 条记录
- 清除浏览器数据会删除历史记录
- 建议定期导出重要文章

## 💡 使用技巧

1. **批量处理**
   - 一次可以处理多篇文章，提高效率
   - 建议每次不超过 10 篇，避免处理时间过长

2. **图片处理**
   - 如果文章图片较多，转换时间会稍长，请耐心等待
   - 建议选择"完整包"下载，确保图片不丢失

3. **历史记录**
   - 定期查看历史记录，可以快速找到之前转换的文章
   - 重要文章建议及时下载保存

4. **文件管理**
   - 下载的 ZIP 文件可以解压后查看
   - Markdown 文件可以用任何文本编辑器打开
   - 推荐使用 Typora、VS Code 等工具查看 Markdown

5. **图片导出**
   - 导出为图片时，可以调整高度来控制内容量
   - 内容较长时会自动分段，无需担心
   - 多张图片会自动打包，方便管理
   - 建议使用 2x 分辨率，平衡清晰度和文件大小
